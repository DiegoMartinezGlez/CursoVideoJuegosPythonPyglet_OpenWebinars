# 2.2 Game Engine - Game Loop

En esta lección se usa el teclado para mover un objeto en pantalla:

- main00.py: ejemplo de movimiento de un objeto en horizontal (eje x) pero sin game loop, no se consigue un movimiento continuo
- main01.py: se añade game loop (función "update") y se gestiona el movimiento continuo cuando si se deja pulsado botón
- main02.py: usa una fuente de texto propia a partir de fichero .ttf
- main03.py: se añade control para el movimiento en vertical (eje y)
